# Demo 缺陷测试基准

> 本文件用于项目验收和效果评估，不属于 Agent 输入。

| 项目 | 内容 |
| --- | --- |
| 基准编号 | GROUND-TRUTH-001 |
| 关联需求 | PRD-FR-003 |
| 需求规则 | 用户注册时，用户名长度必须大于等于 6 位 |
| 实际缺陷 | 后端注册接口未校验用户名长度 |
| 缺陷表现 | 小于 6 位的用户名可以注册成功 |
| 评估用途 | 验证 Agent 能否在未提前获知缺陷答案时发现并形成证据链 |

该文件不得添加到 `ai-testflow.yml` 的 PRD、知识库或 Agent 上下文路径中。
